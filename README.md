# Tripleten web_project_around_pt

Esse projeto tem o objetivo de compartilhar as viagens de um exmplorador através de cards com o nome e a imagem dos locais visitados. Permitindo aos visitantes que curtam os lugares de sua preferência.

Foi construído em uma base HTML, estilizado com CSS e as ações de adicionar novos cards, curtir, excluir cards e edição do perfil do explorador foi feita utilizando JavaScript.
